Contenidos iniciales a JavaScript.


Introducción:

JavaScript es un lenguaje de programación que se ejecuta en el navegador y permite que una página web deje de ser estática para volverse interactiva. Mientras que HTML estructura el contenido y CSS lo diseña, JavaScript se encarga del comportamiento: Responde a acciones del usuario, realiza cálculos, valida datos y modifica elementos en pantalla.

Cuanto trabajos por primera vez con JavaScript es importante comprender cómo se conecta con el documento HTML y cómo interactúa con el navegador.


Vinculación de JavaScript con HTML.

La forma profesional de trabajar es utilizando un archivo externo. Esto permite mantener el código organizado, separado del contenido y más fácil de mantener.

Supongamos que tenemos un arachivo llamado index.html. Dentro de él escribimos la estructura básica de HTML: 

1    <!DOCTYPE html> <!-- Define que el documento es HTML -->
2    <html lang="es"> <!-- Etiqueta principal o raiz que encierra todo el contenido del documento-->
3        <head>  <!-- Contiene información del documento, como el titulo, enlaces a CSS-->
4            <meta charset="UTF-8"> <!-- Define la codificación de caracteres para soportar tildes y ñ-->
5            <link rel="shortcut icon" href="" type="image/x-icon"><!-- Se utiliza para indicar el ícono de la página en la pestaña del navegador y, al dejar la ruta vacía, se evita que el navegador intente buscar un favicon inexistente y genere un error.-->
6            <meta name="viewport" content="width-device-width, initial-scale=1.0"> <!-- Ajusta la visualización en dispositivos móviles--> 
7            <title>Mi primera Documento HTML</JavaScript>   <!-- Define el título que aparece en la pestaña del navegador-->
8        </head>
9        <body> 
10            <h1>Introducción a JavaScript</h1> 
11            <script src="main.js"></script> 
12        </body>
13    </html>

La etiqueta <script> enlaza el archivo JavaScript mediante el atributo src. En este caso el navegador buscará un archivo llamadoo main.js en la misma carpeta.

Es recomendable colocar esa línea justo antes de cerrar el </body>, ya que de esta manera el contenido HTML se carga primero y luego se ejecuta el código JavaScript.


Creación del archivo JavaScript

Una vez declarada la referencia al archivo externo mediante el atributo src, se procede a la creación del archivo main.js.

En el entorno Visual Studio Code, es posible generarlo directamente mediante la combinación Ctrl + Clic sobre el nombre del archivo indicado en el atributo. El editor creará automáticamente el archivo en la ubicación correspondiente, respetando la ruta especificada en el documento HTML.


Introduccion a console.log()

A continuación, se incorpora la primer linea de código con el objetivo de verificar el correcto funcionamiento de la vinculación: 

console.log("Vinculación exitosa");

La instrucción console.log() permite mostrar información en la consola del navegador. En esta instancia, su finalidad es exclusivamente verificadora: confirmar que la integración entre el documento HTML y el archivo JavaScript se ha realizado correctamente.

Para comprobar, se debe abrir el archivo HTML en el navegador, acceder a las herramientas de desarrollador tecla F12 y seleccionar la pestaña Consola. Si el mensaje se visualiza correctamete, la vinculación se encuentra operativa.

Declaración de variables en JavaScript

En JavaScript, para declarar una variable se utilizan palabras reservadas del lenguaje. En la actualidad son let y const.

Uso de let.

La palabra reservada let se utiliza para declarar variables cuyo valor puede cambiar durante la ejecución del programa. Es la forma más común de declarar variables cuando se espera modificar el contenido almacenado. 

1    let nombre = "Juan";
2    let edad = 25;
3    console.log(nombre);
4    console.log(edad);
Consola: 
Juan
25

Uso de const.

La palabra reservada const se utiliza para declarar variables cuyo valor no debe cambiar después de ser asignado. 

1    const PI = 3.1416;
2    console.log(PI);

Si se intenta modificar una variable declara con const, el interprete generará un error. 

1    const PI = 3.1416;
2    PI = 3.14;

Esto produce un error porque las constantes no pueden cambiar su valor.


La consola del Navegador.

La consola constituye una herramienta esencial para el desarrollo. Permite visualizar resultados, analizar el comportamiento del código y detectar posibles errores.

El método estándar para mostrar información es: 

1    console.log("Mensaje de prueba");

También es posible mostrar el contenido de variables: 

1    let curso = "Laboratorio de programación";
2    console.log(curso)

Asimismo, la consola permite evaluar expresiones matemáticas: 

1    console.log( 15 + 5 );
2    console.log( 20 / 4 );

Durante las primeras etapas del aprendizaje, la consola funciona como el principal medio de visualización de resultados.


Tipos de Datos.

JavaScript es un lenguaje de tipado dinámico. El tipo de dato se determina automáticamente según el valor asignado a la variable.

Texto (String).
1    let nombre = "Juan";

Representa secuencias de caracteres delimitadas por comillas dobles.

Número (Number).
1    let edad = 30;
2    let precio = 99.50;

Incluye tanto números enteros como decimales.

Booleano (Boolean).
1    let activo = true;
2    let aprobado = false;

Representa valores lógicos.

Underfined 
1    let dato;
2    console.log(dato);

Indica cuando una variable no ha sido definida.

Null 
1    let valor = null;

Representa la ausencia intencional de valor.

NaN (Not a Number)

En JavaScript existe un valor especial llamado NaN, que significa Not a Number(no es un número).

Este valor aparece cuando el programa intenta realizar una operación matemática con datos que no pueden convertirse correctamente en números.

Por ejemplo, si se intenta dividir texto que no representa un numero, el resultado será NaN. 

1    console.log("hola"/2);

Resultado en la consola: NaN

Representa la ausencia intencional de valor.

Interacción con el Usuario.

JavaScript dispone de funciones integradas que permiten interactuar con el usuario mediante cuadros de diálogo.

Solicitud de datos con prompt()

1 let nombre = prompt("Ingrese su nombre: ");
2    console.log("Bienvenido" + nombre);

La función prompt() devuelve siempre un valor de tipo texto, independientemente del contenido ingresado por el usuario.

El valor que devuelve prompt() siempre es interpretado como texto, es decir, como un dato de tipo string. Esto ocurre incluso cuando el usuario ingresa números. Por esta razón, cuando se necesitan realizar operaciones matemáticas, es necesario convertir explícitamente el dato ingresado a un tipo numérico.

Por ejemplo: 

1    let numero = Number(prompt("Ingrese un número: "));
2    console.log(numero + 10);

En este caso se utiliza la función Number() para transformar el valor ingresado desde texto a número, permitiendo realizar operaciones aritméticas correctamente.

Conversión de tipos de datos

En muchas situaciones es necesario transformar un valor de un tipo de dato a otro. Esto ocurre frecuentemenete cuando los datos son ingresados mediante prompt(), ya que el navegador los interpreta como texto.

JavaScript dispone de diferentes funciones para realizar conversiones explícitas entre tipos de datos.
